- [Memory add](https://docs.mem0.ai/api-reference/memory/add-memories)

- [Batch Update](https://docs.mem0.ai/api-reference/memory/batch-update)

- [Create Memory Export](https://docs.mem0.ai/api-reference/memory/create-memory-export)  
  *Generate an export of stored memories.*

- [Delete Memories](https://docs.mem0.ai/api-reference/memory/delete-memories)  
  *Delete selected memories from the system.*

- [Get Memory](https://docs.mem0.ai/api-reference/memory/get-memory)  
  *Retrieve details of a specific memory.*

- [Get Memory Export](https://docs.mem0.ai/api-reference/memory/get-memory-export)  
  *Access an exported memory file.*

- [History Memory](https://docs.mem0.ai/api-reference/memory/history-memory)  
  *View the history of memory interactions.*

- [Update Memory](https://docs.mem0.ai/api-reference/memory/update-memory)  
  *Update details of an existing memory.*

- [V1 Get Memories](https://docs.mem0.ai/api-reference/memory/v1-get-memories)  
  *Version 1: Retrieve memories.*

- [V1 Search Memories](https://docs.mem0.ai/api-reference/memory/v1-search-memories)  
  *Version 1: Search for memories based on criteria.*

- [V2 Get Memories](https://docs.mem0.ai/api-reference/memory/v2-get-memories)  
  *Version 2: Enhanced retrieval of memories.*

- [V2 Search Memories](https://docs.mem0.ai/api-reference/memory/v2-search-memories)  
  *Version 2: Enhanced search for memories.*
